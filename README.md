## Usage

This bot is triggered by either `!kaamelott` or `!kmlt`

### Asking for help

`!kaamelott help`

### Play a sound

`!kaamelott <sound>`
`!kaamelott random`

## Contributing

```
git clone git@github.com:sofianegargouri/kaamelott-discord-bot.git
cd kaamelott-discord-bot
yarn
echo "DISCORD_TOKEN=<YOUR DISCORD API BOT TOKEN>" > .env
yarn start
```
